1 Motivation

Classical Shannon theory deliberately abstracts away meaning. This work makes semantics operational: information is semantic when it helps the decoder infer a task-relevant hidden state.

Core idea:
encode $X$ once, then reconstruct both $\hat{X}$ for appearance and $\hat{S}$ for meaning.
  • Appearance fidelity is measured by $d_a(X,\hat X)$.
  • Semantic fidelity is measured by the quality of estimating $S$.
  • The encoder observes $X$, not the hidden state $S$.

2 System Model

A memoryless source generates $(S, X) \sim p(s,x)$. The encoder observes only $X^n$ and sends a rate-limited index. The decoder produces both semantic and appearance reconstructions.

3 Main Question

How many bits are needed to preserve both observable appearance and hidden semantic state?

The answer is captured by a two-distortion rate-distortion function involving semantic distortion $D_s$ and appearance distortion $D_a$.

4 SORDF Characterization

The State-Observation Rate-Distortion Function converts semantic reconstruction into an equivalent distortion on the observed variable.

$R(D_s,D_a)=\min_{p(\hat{s},\hat{x}|x)} I(X;\hat{S},\hat{X})$ $\mathrm{s.t.}\quad \mathbb{E}[\hat d_s(X,\hat S)]\le D_s,\quad \mathbb{E}[d_a(X,\hat X)]\le D_a$
  • $S \leftrightarrow X \leftrightarrow \hat{S}$ makes the semantic loss computable through $X$.
  • Tensorization extends the one-shot equivalence to block coding.
  • The problem becomes lossy coding with two coupled constraints.

8 Expected Evidence

  • Lower label error at matched reconstruction distortion.
  • Clearer trade-off curve between $D_a$ and $D_s$.
  • Largest gains in the low-rate bottleneck regime.
$D_s$↓ semantic error
$D_a$↓ appearance loss
$R$↓ coding rate

Takeaway

Semantic coding is not simply compression followed by a classifier. The rate allocation itself changes when meaning and appearance are optimized together.
