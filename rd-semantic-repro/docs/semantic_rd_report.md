## 3. 理论推导

### 3.1 问题建模

**信源模型**：(S, X) ~ p(s, x)，i.i.d. 序列 (Sⁿ, Xⁿ)

**编码器**：fₙ: Xⁿ → {1, ..., 2ⁿᴿ} = W
**解码器**：gₙ: W → Ŝⁿ × X̂ⁿ

**失真度量**：
- 语义失真：dₛ(sⁿ, ŝⁿ) = (1/n) Σᵢ dₛ(sᵢ, ŝᵢ)
- 表象失真：dₐ(xⁿ, x̂ⁿ) = (1/n) Σᵢ dₐ(xᵢ, x̂ᵢ)

**achievability**：(R, Dₛ, Dₐ) 可达，当且仅当存在编码-解码序列使得：
```
lim E[dₛ(Sⁿ, Ŝⁿ)] ≤ Dₛ
lim E[dₐ(Xⁿ, X̂ⁿ)] ≤ Dₐ
```

### 3.2 SORDF 

$$ R(Dₛ, Dₐ) = min I(X; Ŝ, X̂)$$
$$
    E[d̂ₛ(X, Ŝ)] ≤ Dₛ \\
    E[dₐ(X, X̂)] ≤ Dₐ
$$
等价语义失真：
$$ d̂ₛ(x, ŝ) = (1/p(x)) Σₛ p(s, x) dₛ(s, ŝ) = E_{S|x}[dₛ(S, ŝ)] $$

**proof**：

1. **Markov 链**：Sⁿ ↔ Xⁿ ↔ Ŝⁿ
2. **等价转换**：
  $$ E[dₛ(Sⁿ, Ŝⁿ)] = Σ_{xⁿ, ŝⁿ} p(xⁿ, ŝⁿ) · (1/n) Σᵢ d̂ₛ(xᵢ, ŝᵢ) $$

### 3.3 联合高斯模型（Section IV）

**模型设定**：
- S, X 联合高斯，零均值，协方差矩阵 Σ
- 失真度量：均方误差 dₛ = ‖s - ŝ‖², dₐ = ‖x - x̂‖²

**条件分布**：
$$
S|X=x \sim N(Σ_{SX}Σ_X⁻¹x, Σ_S - Σ_{SX}Σ_X⁻¹Σ_{SX}ᵀ)
$$

**等价语义失真**
$$
d̂ₛ(x, ŝ) = tr[Σ_S - Σ_{SX}Σ_X⁻¹Σ_{SX}ᵀ] + ‖Σ_{SX}Σ_X⁻¹x - ŝ‖² \\
           = mmse + ‖Σ_{SX}Σ_X⁻¹x - ŝ‖²
$$

其中 mmse 是观测 X 后估计 S 的最小均方误差。

**SORDF** ：
$$
R(Dₛ, Dₐ) = min I(X; Ŝ, X̂) \\
s.t. \  E‖Σ_{SX}Σ_X⁻¹X - Ŝ‖² ≤ Dₛ - mmse \\
      E‖X - X̂‖² ≤ Dₐ
$$

#### 3.3.1 标量情形

当 S, X 均为标量时：
$$
R(Dₛ, Dₐ) = (1/2) max{ (log(Σ_X/Dₐ))⁺, (log(Σ_{SX}²/(Σ_X(Dₛ-mmse))))⁺ }
$$

#### 3.3.2 向量情形

**特殊模型**：X = 1ₘS + Z，其中 S ~ N(0, σₛ²), Z ~ N(0, σ_Z²I)


### 4.4 二元分类模型（Section V）

**模型设定**：
- S ~ Bernoulli(1/2)
- X|S=0 ~ N(A, σ²), X|S=1 ~ N(-A, σ²)
- 边际分布：高斯混合
- dₛ：汉明失真，dₐ：平方误差

#### 4.4.1 仅语义约束（Proposition 5）

```
R(Dₛ, ∞) = 1 - (1/2)∫[N⁺(x) + N⁻(x)]h₂(g(x))dx
```

其中：
```
g(x) = [1 + exp(λ(1 - e^{-2Ax/σ²})/(1 + e^{-2Ax/σ²}))]⁻¹
```

**关键洞察**：
- g(x) 是**软决策**（soft decision），非硬阈值
- 当 x >> 0 时，g(x) ≈ 1（强烈暗示 S=0）
- 当 x ≈ 0 时，g(x) ≈ 1/2（信息最少）
- **"局部分类+压缩"方案次优**（除极端情况 Dₛ = Q(A/σ)）

这与高斯情形形成鲜明对比：在高斯情形下，先 MMSE 估计再压缩是最优的（Proposition 2），但在分类问题中，**软加权后验信念**优于硬贝叶斯分类。

#### 4.4.2 双重约束上界（Proposition 6）

**编码方案**：
1. 先用 Proposition 5 将 X 编码为 Ŝ，速率 R(D, ∞)，满足语义约束 D
2. 条件于 Ŝ，编码残差 X - E[X|Ŝ] 为 X̃，使用高斯码本，速率 (1/2)(log(η/Dₐ))⁺
3. 解码器重建 X̂ = X̃ + E[X|Ŝ]

**优化问题**：
```
R(Dₛ, Dₐ) ≤ min_{D ∈ [Q(A/σ), Dₛ]} { R(D, ∞) + (1/2)(log(η/Dₐ))⁺ }
```

其中 η 是条件残差的方差，依赖于软决策函数 g_D(x)。

## 6. 实验设置

### 6.1 实验设计概述

本文以**理论推导**为主，数值实验用于验证和可视化理论结果。主要包含三类实验：

### 6.2 实验一：高斯向量 SORDF 可视化

**设置**：
- 模型：X = 1ₘS + Z
- 参数：m = 维度, σₛ², σ_Z²
- 输出：R(Dₛ, Dₐ) 的三维曲面及等高线图

**结果**（图 3）：
- 左图：R(Dₛ, Dₐ) 随 Dₛ, Dₐ 增加而单调递减的曲面
- 右图：等高线图清晰显示五个区域的分界
- **关键观察**：区域 S₄ 中，Dₛ 和 Dₐ 存在明确的权衡（tradeoff）——改善一方会恶化另一方

### 6.3 实验二：分类问题软决策函数

**设置**：
- A = σ² = 1, 因此 Q(A/σ) = Q(1) ≈ 0.1587
- 考察不同 Dₛ 下的 g(x) 形态

**结果**（图 4 左）：
- Dₛ = 0.50（最宽松）：g(x) ≡ 0.5（完全随机猜测，不传输信息）
- Dₛ = 0.39：平缓的 Sigmoid 型曲线
- Dₛ = 0.27：更陡峭的过渡
- Dₛ = 0.16（最严格）：接近阶跃函数（硬阈值）

**洞察**：失真约束越严格，决策越"硬"；但即使 Dₛ = 0.16，g(x) 仍非理想阶跃函数。

### 6.4 实验三：分类问题速率-失真曲线

**设置**：
- A²/σ² = 10（高信噪比）
- 比较三种方案：
  1. 本文上界（Proposition 6）
  2. 理想情形：编解码器均知 S，R = (1/2)log(σ²/Dₐ)
  3. 朴素方案：直接编码 X，R = (1/2)log((A²+σ²)/Dₐ)

**结果**（图 5）：
- 固定 Dₛ 时，R 随 Dₐ 增加而递减
- 本文方案显著优于朴素方案（尤其 Dₐ 小时）
- 与理想方案仍有差距（因 S 未知）
- **不同 Dₛ 的曲线**：Dₛ 越小，曲线越高（需要更多比特保护语义信息）

### 6.5 实验结果的理论意义

| 实验 | 验证的理论命题 | 核心发现 |
|------|---------------|----------|
| 高斯向量可视化 | Proposition 4 | 五个区域清晰划分，权衡区域 S₄ 存在非平凡交互 |
| 软决策函数 | Proposition 5 | 最优决策是软的，硬贝叶斯分类次优 |
| 速率-失真曲线 | Proposition 6 | 语义约束显著影响可达速率，联合优化优于分离方案 |
