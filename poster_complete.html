Semantic Rate-Distortion: From Theory to Deep Learning

Numerical Reproduction of Gaussian & Binary Case Studies  |  RDPC Extension on MNIST with Perception & Classification Constraints
Based on: Liu, Zhang & Poor (IEEE ISIT 2021)  &  Task-Oriented Lossy Compression (IEEE JSAC 2025)
1. Problem Formulation: The (S, X) Semantic Source

A memoryless information source is modeled as a tuple of random variables $(S, X) \sim p(s, x)$:

$S$Intrinsic semantic state — not directly observable (e.g., digit label) $X$Extrinsic appearance observation — what the encoder sees (e.g., image) $\hat S$Reconstructed semantic estimate at decoder $\hat X$Reconstructed appearance at decoder

Core Markov Chain: $S \leftrightarrow X \leftrightarrow (\hat S, \hat X)$

The encoder only observes $X$; the decoder must reconstruct both $\hat S$ and $\hat X$ under two distortion constraints:

  • Semantic distortion: $d_s(s, \hat s)$ — e.g., Hamming distance, cross-entropy
  • Appearance distortion: $d_a(x, \hat x)$ — e.g., squared error, MSE
2. State-Observation Rate-Distortion Function (SORDF)

The fundamental limit is the SORDF:

$$R(D_s, D_a) = \min_{p(\hat s, \hat x|x)} I(X; \hat S, \hat X)$$ $$\text{s.t. } \mathbb{E}\,\hat d_s(X, \hat S) \leq D_s,\quad \mathbb{E}\,d_a(X, \hat X) \leq D_a$$

Theorem 1 (Equivalent Distortion): The unobservable semantic distortion can be transformed into an observable form:

$$\hat d_s(x, \hat s) = \frac{1}{p(x)}\sum_s p(s, x)\, d_s(s, \hat s) = \mathbb{E}[d_s(S, \hat s) \mid X = x]$$

Key Properties (Proposition 1):

  • Monotonicity $R(D_s, D_a)$ decreases as $D_s$ or $D_a$ increase
  • Convexity $R(D_s, D_a)$ is jointly convex in $(D_s, D_a)$
  • Boundary $R = 0$ when constraints are loose enough
3. Binary Classification: $R(D_s, \infty)$ (Prop. 5)

Model: $S \sim \text{Bernoulli}(1/2)$,  $X|S=0 \sim \mathcal{N}(A, \sigma^2)$,  $X|S=1 \sim \mathcal{N}(-A, \sigma^2)$

Optimal soft decision:

$$g(x) = P(\hat S = 0 \mid X = x) = \left[1 + \exp\!\left(\lambda \cdot \frac{1 - e^{-2Ax/\sigma^2}}{1 + e^{-2Ax/\sigma^2}}\right)\right]^{-1}$$

Rate: $R(D_s, \infty) = 1 - \frac{1}{2}\int_{-\infty}^{\infty}[N^+(x) + N^-(x)]\, h_2(g(x))\,dx$

$\lambda < 0$ is solved numerically via bisection from the constraint $\int (N^+ - N^-)g\,dx = 1 - 2D_s$.

  • $D_s = Q(A/\sigma)$ → $R = 1$ bit (Bayes-optimal hard decision)
  • $D_s = 0.5$ → $R = 0$ bits (random guessing)
  • As $D_s$ relaxes, $g(x)$ flattens — less certainty needed
4. Finite Appearance: $R(D_s, D_a)$ Upper Bound (Prop. 6)

A two-stage achievable scheme:

$$R(D_s, D_a) \leq \min_{D \in [Q(A/\sigma),\, D_s]} \left\{ R(D, \infty) + \frac{1}{2}\left[\log\frac{\eta}{D_a}\right]^+ \right\}$$
  1. Semantic encoding: compress $X \to \hat S$ at rate $R(D, \infty)$
  2. Appearance refinement: encode residual $X - \mathbb{E}[X|\hat S]$ with Gaussian codebook

$\eta = \text{Var}(X \mid \hat S)$ is computed via numerical integration over $g_D(x)$.

5. Scalar Gaussian SORDF (Prop. 3)

For $(S, X)$ jointly Gaussian (scalar), squared-error distortions:

$$R(D_s, D_a) = \frac{1}{2}\max\!\left\{ \left[\log\frac{\Sigma_X}{D_a}\right]^+,\; \left[\log\frac{\Sigma_{SX}^2}{\Sigma_X(D_s - \text{mmse})}\right]^+ \right\}$$

where $\text{mmse} = \mathbb{E}[(S - \mathbb{E}[S|X])^2]$ is the irreducible estimation error.

Two regimes:

  • Appearance-dominated: $\frac{D_a}{\Sigma_X} < \frac{D_s - \text{mmse}}{\Sigma_{SX}^2/\Sigma_X}$ → rate set by appearance
  • Semantic-dominated: the opposite → rate set by semantic requirement
6. Vector Gaussian: Five-Region Structure (Prop. 4)

Model: $X = \mathbf{1}_m S + Z$, where $S \sim \mathcal{N}(0, \sigma_S^2)$, $Z \sim \mathcal{N}(0, \sigma_Z^2 I_m)$.

Apply orthogonal transform to diagonalize: $m$ independent Gaussian channels, only one carries semantic information. The distortion-allocation problem yields five distinct regions:

RegionConditionPhysical Meaning
S1: Zero-rateBoth constraints looseNo encoding needed
S2: Appearance-drivenAppearance constraint tightAll $m$ channels encoded
S3: Semantic-drivenSemantic constraint tightOnly semantic channel encoded
S4: Mixed boundaryBoundary interactionOne constraint dominates at margin
S5: TRADEOFFGenuine competitionRate must be split across both goals
7. From SORDF to RDPC: The Deep Learning Extension

The 2025 JSAC paper extends classical rate-distortion with three simultaneous constraints:

$$R(D, P, C) = \min_{p_{\hat X|X}} I(X; \hat X)$$ $$\begin{aligned} \text{(1a) Distortion:}&\quad \mathbb{E}[\Delta(X, \hat X)] \leq D \\ \text{(1b) Perception:}&\quad d(p_X, p_{\hat X}) \leq P \quad\text{(Wasserstein-1)} \\ \text{(1c) Classification:}&\quad H(S \mid \hat X) \leq C \quad\text{(Information Bottleneck)} \end{aligned}$$

Key Insight — Hamming → Cross-Entropy Degeneration:

Replacing Hamming distortion $d_s(s,\hat s) = \mathbf{1}[s \neq \hat s]$ with cross-entropy loss degenerates the semantic constraint into the Information Bottleneck form:

$$\mathcal{L}_{\text{CE}}(S, \hat S) = -\sum_{c} \mathbf{1}[S=c] \log p(\hat S = c \mid X)$$

When $\lambda_p = 0$, the framework reduces exactly to Variational Information Bottleneck (VIB).

8. RDPC on MNIST: Deep Architecture

System Pipeline:

$S$ (label) → $X$ (28×28) → Encoder → $Z \in \mathbb{R}^k$ → Decoder → $\hat X$ → Classifier → $\hat S$
            $\uparrow$
           Discriminator (WGAN-GP)

NetworkArchitectureOutput
Encoder Conv 32→64→128 + FC $Z \in \mathbb{R}^k$
Decoder FC + ConvT 128→64→32→1 $\hat X \in [0,1]^{28\times28}$
Discriminator Conv 32→64→128 + FC→1 (no sigmoid) Scalar critic score
Classifier Conv 32→64→128 + Pool + FC→10 10-class logits (frozen)

Rate Estimation: Since $I(X; Z)$ is intractable for continuous $Z$, we use the Gaussian entropy upper bound:

$$R \approx \frac{1}{2}\sum_{i=1}^{k} \log_2(1 + \sigma_i^2)$$

The bottleneck dimension $k$ implicitly controls the rate — smaller $k$ = stronger compression.

9. Loss Function & Alternating Training

Joint Lagrangian relaxation:

$$\mathcal{L}_{\text{train}} = \underbrace{\lambda_d \cdot \text{MSE}(X, \hat X)}_{\text{Distortion (1a)}} \;-\; \underbrace{\lambda_p \cdot \mathbb{E}[D(\hat X)]}_{\text{Perception (1b)}} \;+\; \underbrace{\lambda_c \cdot \text{CE}(S, \hat S)}_{\text{Classification (1c)}}$$

WGAN-GP alternating schedule:

StepNetworksLoss
Critic (×5) Discriminator $W_1 = \mathbb{E}[D(\hat X)] - \mathbb{E}[D(X)] + 10 \cdot \text{GP}$
Generator (×1) Encoder + Decoder $\lambda_d \text{MSE} - \lambda_p \mathbb{E}[D(\hat X)] + \lambda_c \text{CE}$

Gradient Penalty (GP): enforces 1-Lipschitz constraint on the critic via random interpolations $\tilde x = \epsilon x + (1-\epsilon)\hat x$:

$$\text{GP} = \mathbb{E}_{\tilde x}\left[(\|\nabla_{\tilde x} D(\tilde x)\|_2 - 1)^2\right]$$

Classifier Pretraining: 5 epochs on real MNIST → 96.55% test accuracy, then frozen.
